# Taxonomy Overview: Leaf Nodes and Example Questions

**Generated**: December 8, 2025  
**Verified & Corrected**: December 22, 2025  
**Purpose**: Map each leaf node from the Six Pillars taxonomies to concrete example questions and framework coverage.

---

## How to Use This Document

This overview takes every **leaf node** (terminal concept) from the six selected taxonomies and provides:
1. **Example Question**: A concrete question that testing this capability would answer.
2. **Framework Coverage**: Which of the 6 Pillars encompass this capability.
3. **Task Layer**: Where this maps to the benchmark (Layer 1, 2, or 3).

**Framework Key:**
- **P1** = Pillar 1: Gaver (Ecological/Physics of Events)
- **P2** = Pillar 2: Bregman (Auditory Scene Analysis)
- **P3** = Pillar 3: Rauschecker (Dual Stream Neuroscience)
- **P4** = Pillar 4: Katz (Buffalo Model of APD)
- **P5** = Pillar 5: Atkinson-Shiffrin (Modal Model of Memory)
- **P6** = Pillar 6: Baddeley (Working Memory Model)
